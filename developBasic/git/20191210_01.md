## git搭建

> 在官网下载git，并安装，参考链接 [Git下载、安装与环境配置](https://blog.csdn.net/huangqqdy/article/details/83032408)

在文件夹，右键空白处，在弹出来的菜单中，点击  `Git Bash Here`


## git管理

> 拉取代码

```nginx
$ git clone https://***.git
```

> 更新代码

```nginx
$ git pull
```

> 提交代码

```nginx
$ git commit -m ""
$ git push
```
