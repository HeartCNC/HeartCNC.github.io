vue 运行报错 Cannot call .tap() on a plugin that has not yet been defined. Call plugin('preload').use(<Plugin>) first.

2022年7月6日
  • vue-cli

Vue Cli 配置多入口报错解决办法

ERROR  Error: Cannot call .tap() on a plugin that has not yet been defined. Call plugin('preload').use(<Plugin>) first.
Error: Cannot call .tap() on a plugin that has not yet been defined. Call plugin('preload').use(<Plugin>) first.
    at Object.tap 
    ...
    ...
error Command failed with exit code 1.
info Visit https://yarnpkg.com/en/docs/cli/run for documentation about this command.

Vue CLI 当在 multi-page 模式下构建时，webpack 配置会包含不一样的插件 (这时会存在多个 html-webpack-plugin 和 preload-webpack-plugin 的实例)。如果你试图修改这些插件的选项，请确认运行 vue inspect。

解决方法​

chainWebpack(config) {
  config.plugin("preload-index").tap(() => [
    {
      rel: "preload",
      fileBlacklist: [/\.map$/, /hot-update\.js$/, /runtime\..*\.js$/],
      include: "initial",
    },
  ])
}
