vscode 常用快捷键速览

2022年6月23日
  • vscode

代码操作​

  • Ctrl + D 选中当前单词
  • Ctrl + L 选中当前行
  • Ctrl + G 跳转到指定行

插入​

  • Ctrl + Shift + Enter 向上插入一空行
  • Ctrl + Enter 向下插入一空行

移动​

  • Alt + Up 向上移动
  • Alt + Down 向下移动

复制​

  • Alt + Shift + Up 向上复制一行
  • Alt + Shift + Down 向下复制一行

编辑器​

展示​

  • Ctrl + \\ 分屏
  • Ctrl + Shift + [ 折叠代码块
  • Ctrl + Shift + ] 展开代码块
  • Ctrl + [ 向左缩进
  • Ctrl + ] 向右缩进

光标​

  • Ctrl + Home 移到文件头
  • Ctrl + Eed 移到文件尾
  • F8 跳到下一个 "问题"
