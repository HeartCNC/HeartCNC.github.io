UI框架整理

2022年6月24日

    常用UI框架整理，包含移动端、PC端管理后台、微信小程序端、图标等等

    移动端​

    Vant​

    Vant 是一个轻量、可靠的移动端组件库。

    👉 https://youzan.github.io/vant/v2/#/zh-CN/

    Mint UI​

    Mint UI是 饿了么团队开发基于vue .js的移动端UI框架，它包含丰富的 CSS 和 JS 组件，能够满足日常的移动端开发需要。

    👉 http://mint-ui.github.io/#!/zh-cn

    管理端​

    Element​

    Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的组件库，提供了配套设计资源，帮助你的网站快速成型。由饿了么公司前端团队开源。

    👉 https://element.eleme.cn/

    iView​

    iView 是一套基于 Vue.js 的开源 UI 组件库，主要服务于 PC 界面的中后台产品。

    👉 https://iview.github.io/

    Muse-UI​

    基于 Vue 2.0 优雅的 Material Design UI 组件库

    👉 https://muse-ui.org/

    响应式​

    Bootstrap​

    简洁、直观、强悍的前端开发框架，让web开发更迅速、简单。

    👉 https://www.bootcss.com/

    微信小程序​

    Vant​

    Vant 是一个轻量、可靠的移动端组件库。

    👉 https://vant-contrib.gitee.io/vant-weapp/#/home

    iView​

    一套高质量的微信小程序 UI 组件库

    👉 https://weapp.iviewui.com

    图标​

    Font Awesome​

    一套绝佳的图标字体库和CSS框架

    👉 https://fontawesome.dashgame.com/

    Iconfont​

    阿里妈妈MUX倾力打造的矢量图标管理、交流平台。

    设计师将图标上传到 iconfont 平台，用户可以自定义下载多种格式的icon，平台也可将图标转换为字体，便于前端工程师自由调整与调用。

    👉 https://www.iconfont.cn/
