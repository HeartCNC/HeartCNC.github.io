ChainedSet​

方法​

  • add(value) 添加/追加 给Set末尾位置一个值
  • prepend(value) 添加 给Set开始位置一个值
  • clear() 清除
  • delete(value) 删除某个值
  • has(value) 检测Set中是否存在一个值
  • values(): Array 返回Set中值的数组
  • merge(arr: Array) 连接给定的数组到 Set 尾部
  • batch(handler: (chain: ChainedSet) => void) 对当前配置上下文执行函数
  • when(condition: boolean, whenTruthy: chain: ChainedSet => void, whenFalsy: chain: ChainedSet => void) 条件执行一个函数去继续配置，当条件为真，调用whenTruthy，当条件为假，调用whenFalsy
