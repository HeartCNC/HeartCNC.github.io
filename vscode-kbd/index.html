代码操作​

插入​

Ctrl + Shift + Enter 向上插入一空行Ctrl + Enter 向下插入一空行

移动​

Alt + Up 向上移动Alt + Down 向下移动

复制​

Alt + Shift + Up 向上复制一行Alt + Shift + Down 向下复制一行

Ctrl + \\ 分屏Ctrl + Shift + [ 折叠代码块Ctrl + Shift + ] 展开代码块Ctrl + [ 向左缩进Ctrl + ] 向右缩进

展示​
